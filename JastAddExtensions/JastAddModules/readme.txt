Make sure that you have the following directories in the source zip:

	IntertypeDeclarations
	JastAdd
	JastAddModules
	Java1.5Backend
	Java1.5Frontend
	Java1.4Backend
	Java1.4Frontend
	
To perform a build, do:

cd JastAddModules
ant clean
ant

The JHotdraw 7.1 case study can be found in JastAddModules/case_study/JHotdraw 7.1/
See that directory for additional instructions on how to build the case study examples.
